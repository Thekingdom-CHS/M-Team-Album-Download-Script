# M-Team-Album-Download-Script
# M-Team 片单批量下载助手


自动按顺序点击 M-Team 片单详情页中的所有下载按钮，支持进度保存、断点续点、实时编号与进度显示、中途暂停/恢复等功能。

## ✨ 功能特性

- **自动批量点击** – 按表格顺序依次点击每个种子的下载按钮
- **断点续点** – 刷新页面或关闭浏览器后，进度自动保存，可从上次中断处继续
- **实时进度显示** – 面板展示总数、已完成数量、当前处理的编号
- **自定义起始位置** – 支持从任意序号开始下载（例如从第 10 个开始）
- **暂停 / 恢复** – 中途可随时暂停，后续可恢复继续
- **间隔可调** – 自定义两次点击之间的延迟时间（默认 30 秒，避免服务器压力）
- **序号标记** – 在每条种子的标题链接前自动添加序号（金黄色圆底），一目了然
- **高亮与滚动** – 点击当前按钮时自动滚动到视野内并高亮边框
- **状态持久化** – 使用 `localStorage` 存储起始索引和已点击数量
- **折叠式面板** – 默认显示一个小图标，点击展开完整控制面板，可随时收起

3. 访问 M-Team 的任意片单详情页（例如 `https://*.m-team.*/album/xxx`），脚本自动生效。

## 🚀 使用指南

1. 打开目标片单页面，等待页面完全加载。
2. 页面**右上角**会出现一个"M-team LOGO"，点击展开控制面板。
3. 根据需要调整 **间隔(ms)**（例如默认的 30000 表示 30 秒）。
4. 点击 **▶ 继续 (断点续点)** 开始自动点击，或点击 **🔢 从第 N 个开始** 自定义起始位置。
5. 点击 **⏸ 暂停** 可随时中断，再次点击 **▶ 恢复** 继续。
6. 所有按钮点击完成后，面板会提示完成，进度自动保留。

### 控制面板按钮说明

| 按钮 | 功能 |
|------|------|
| ▶ 继续 (断点续点) | 从上次保存的进度继续点击 |
| 🔄 从头开始 | 重置进度，从第一个按钮开始点击 |
| ⏸ 暂停 / ▶ 恢复 | 暂停或恢复正在进行的任务 |
| 🗑 重置进度 | 仅清除保存的进度记录，不影响已点击过的按钮 |
| 🔢 从第 N 个开始 | 弹出对话框，输入起始序号（1‑N），覆盖原进度并开始 |
| ⟳ 刷新统计 | 重新获取页面中的下载按钮数量，并刷新序号显示 |

## ⚠️ 注意事项

- 本脚本仅模拟点击下载按钮，实际下载行为（是否弹出新窗口、是否直接下载）由 M-Team 网站决定。
- 为避免造成服务器额外开销，请合理设置点击间隔（建议 ≥ 30000 毫秒）。
- 如果片单包含分页或“加载更多”按钮，请**先手动展开所有条目**，再启动脚本。
- 脚本基于页面结构选择器 `button.ant-btn-icon-only.mr-2` 定位下载按钮，若网站改版可能导致失效。
- 刷新页面后，暂停状态不会保留（任务会自动停止），但已点击进度会被保存。
- 如果点击“从第 N 个开始”，之前的进度将被覆盖（从 N 开始视为前 N-1 个已完成）。

## 🛠️ 技术实现

- 使用 `MutationObserver` 等待页面按钮加载完成。
- 通过 `localStorage` 存储 `起始索引` 和 `已点击数量`，实现断点续点。
- 在每行种子的标题超链接内动态插入 `<span>` 元素显示序号。
- 异步循环控制点击，支持暂停/恢复标志位。
- 面板使用纯 HTML/CSS 实现，无外部依赖。
